MongoDB下Distinct结果集超过16M的解决方案

MongoDB单文档大小不能超过16M，当一个集合数据量很大时（遇到的耽单个集合200W条文档），对其进行Distinct就会出现错误提示：
Array ( [errmsg] => exception: distinct too big, 16mb cap [code] => 10044 [ok] => 0 )

解决方案：
MongoDB的aggeregate聚合方法:

db.myCollection.aggregate( {$group : {_id : "$distinctfield"} }, {$group: {_id:1, count: {$sum : 1 }}});
